## 特性

* [x] 开放源代码
* [x] 不再使用用户名密码登录 OpenList，改为使用令牌，降低敏感信息泄露风险
* [x] 打包后为单文件分发
* [x] 支持 Linux 全平台处理器架构
* [x] 通过 [GitHub Actions](https://docs.github.com/zh/actions) 自动构建 Docker 镜像并推送到 [DockerHub](https://hub.docker.com/r/dr34m/tao-sync)
* [x] 执行失败的任务项支持查看原因
* [x] 执行成功的任务项会自动删除 OpenList 中的任务记录

## 优化

* [x] 强化安全设计，首次运行时自动生成随机密钥，不再写死加密密钥
* [x] 用户校验改为优先使用内存缓存，避免每次请求都读取磁盘数据库
* [x] 版本号改为打包构建时自动从 `version.txt` 读取
* [x] 作业详情中的任务进度样式已针对小屏设备做显示优化
